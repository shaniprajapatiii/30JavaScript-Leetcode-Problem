# 30 Days of JavaScript – LeetCode Challenge

Welcome to the **30 Days of JavaScript** LeetCode challenge! This repository contains solutions to the problems from the [LeetCode 30 Days of JavaScript Study Plan](https://leetcode.com/studyplan/30-days-of-javascript).

---

## 📚 Challenge Overview

- **LeetCode Study Plan:** [30 Days of JavaScript](https://leetcode.com/studyplan/30-days-of-javascript)
- **GitHub Repo:** [shaniprajapatiii/30JavaScript-Leetcode-Problem](https://github.com/shaniprajapatiii/30JavaScript-Leetcode-Problem)
- **Goal:** Solve one JavaScript problem each day for 30 days to improve your coding skills and problem-solving abilities.

---


## 🗂️ Repository Structure

Each day/problem is organized in its own folder for clarity. For example:

- `D1_Closures/` – Day 1: Closures (contains `CreateHelloWorldFunction.js`)
- `D2_BasicArrayTransformations/` – Day 2: Basic Array Transformations
- `D3_FunctionTransformations/` – Day 3: Function Transformations
- `D4_PromisesAndTime/` – Day 4: Promises and Time
- `D5_JSON/` – Day 5: JSON
- `D6_Classes/` – Day 6: Classes

Each folder contains the solution(s) for that day's problem(s).

## 🖥️ How to Run & See Output

To view the output of each solution, follow these steps:

### 1. Using Node.js (Recommended)

1. **Install Node.js** if you haven't already: [Download Node.js](https://nodejs.org/)
2. **Open a terminal** in this project directory.
3. **Navigate to the folder for the day/problem:**

  ```sh
  cd D1_Closures
  ```

4. **Run the desired file:**

  ```sh
  node CreateHelloWorldFunction.js
  ```

  Replace the folder and filename with the problem you want to test.

### 2. Adding Output Code

To display the output for each solution, make sure you include a `console.log` statement at the end of your solution file. For example:

```js
function createHelloWorld() {
  return "Hello World";
}

console.log(createHelloWorld()); // Output: Hello World
```

- **Tip:** Always add a `console.log` to show the function's output for easy testing.

### 3. Using Online Editors

You can also copy the code into online editors like [JSFiddle](https://jsfiddle.net/), [CodePen](https://codepen.io/), or [LeetCode Playground](https://leetcode.com/playground/) to see the output.
